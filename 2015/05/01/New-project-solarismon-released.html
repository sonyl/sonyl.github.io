New project 'solarismon' released

01 May 2015

Recently I released my new homeautomation project solarismon. Its main goal is to monitor the behavior and afterwards optimize the settings of a Rotex Solaris RPS3 solar heating system.

Find the project in its GitHub respository. More information about the project is available on the project main site.
